#PySpider
  基于Flask的爬虫系统，提供落网的数据抓取、存储及简易分析功能
  
##Features
*  爬虫规则配置    
*  资源抓取功能   
*  静态资源存储   
*  抓取结果展示   
*  期刊感谢人数图表展示   
*  数据导出
